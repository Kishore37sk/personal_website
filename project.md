## 🔧 Backend & Frameworks

### 6. **Bike Service Application** 🏍️
- **Repository:** https://github.com/Kishore37sk/Bike-Service-Application
- **Language:** Python (Django)
- **Description:**
  - Complete bike service booking and management application
  - Role-based system for customers and bike station owners
  - **Customer Features:**
    - Register with email and mobile number
    - Book services at specific dates
    - View booking status
    - Access previous bookings history
  - **Owner Features:**
    - Create, edit, delete services and details
    - View all bookings (pending, ready, completed)
    - Mark bookings as ready for delivery
    - Mark bookings as completed
  - **Technology Stack:**
    - Framework: Django
    - Frontend: HTML, CSS, JavaScript, Bootstrap
    - Backend: Python
    - Database: MySQL
  - **Database Schema:**
    - Customer table (booking details and history)
    - Machine table (service data)
    - Owner table (user data)
  - **Setup Instructions:**
    - Python 3.7.6 required
    - Install Django 3.0.5 and django-widget-tweaks
    - Run migrations and start server
    - Access at http://127.0.0.1:8000/
